# 02_sum_of_n_numbers.py - Simple Explanation

## What this program does
This program finds the sum of numbers in different ways.

It can:
- add all numbers from 1 to n
- add only even numbers up to n
- add only odd numbers up to n

## How it works
The program uses loops to keep adding numbers one by one.

### Example
If you enter 5, then:
- sum of first 5 numbers = 1 + 2 + 3 + 4 + 5 = 15

## Very simple way to understand it
This is like standing in a line of numbers and adding them one by one.

Instead of doing it manually, the program does it for you.

## How the solution is made
The program uses a variable called total to store the answer.

1. Start with total = 0
2. Take each number one by one
3. Add it to total
4. Keep going until the end

That is why it is called a loop.

## Program logic: how the program solves the problem
This program solves the problem by collecting numbers step by step.

1. It gets the value of n from the user.
2. It starts the answer at 0.
3. It checks each number from 1 to n.
4. It adds every number to the answer.
5. At the end, it shows the final total.

For even numbers, it only adds numbers like 2, 4, 6, and so on.
For odd numbers, it only adds numbers like 1, 3, 5, and so on.

So the program is solving the problem by adding only the numbers that match the rule.

## What the program shows
It prints:
- sum using a loop
- sum using a formula
- sum of even numbers
- sum of odd numbers

## How to run it
Run this command:

```bash
python3 02_sum_of_n_numbers.py
```

## Easy summary
This program is a number adder. It keeps adding numbers until it reaches the end.
