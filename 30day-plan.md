## 30-days Study Plan:
    **Week 1** : Basics (Syntax, variables, data types, loops, conditionals, functions)
    **Week 2** : Data Structures (lists, dicts, tuples, sets), files I/O, modules
    **Week 3** : OOP Basics, exception handling, working with popular libraries
    **Week 4** : Build 3-4 Projects (Web Scraper, automation script, data analyzer, simple games)

## Key to Success
    - **Daily Practice** (2-3 hours minimum)
    - **Code along, don't just watch** tutorials
    - **Build real projects** - automate something boring, scrape a website, anlyze data
    - **User interactive platforms** - Codecademy, LeetCode, HackerRank, Python.org tutorials
    - **Read other people's code** on GitHub
    - **Join Python communities** - r/learnpython, stack Overflow

## Project Idas for Practice:
    - To-Do list application 
    - Simple game (tic-tac-toe, Hangama, Snake)
    - Web Scraper (new headlines, weather data)
    - Data Analyzer (CSV file processing)
    - Automation scripts (file organizer, email sender)
    - Calculator or Unit Converter
